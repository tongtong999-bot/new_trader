# v5.2 策略交易逻辑详解（大白话版）

## 一、策略的核心思想

**v5.2 是一个"追涨杀跌"的趋势跟踪策略**，简单说就是：
- **上涨趋势时做多**（买涨）
- **下跌趋势时做空**（卖跌）
- **震荡行情时不做**（观望）

## 二、策略怎么判断市场状态？

### 1. 看三个时间周期

策略会同时看三个时间周期的K线图：
- **15分钟图**：看当前价格细节
- **1小时图**：判断入场时机
- **4小时图**：判断大趋势方向

### 2. 判断大趋势（4小时图）

策略用两条均线（EMA20 和 EMA100）来判断大趋势：

**上涨趋势（牛市）**：
- EMA20 在 EMA100 上方
- 说明：短期均线在长期均线之上，市场在上涨

**下跌趋势（熊市）**：
- EMA20 在 EMA100 下方
- 说明：短期均线在长期均线之下，市场在下跌

**震荡行情**：
- EMA20 和 EMA100 反复交叉
- 说明：市场没有明确方向，价格在横盘

### 3. 判断入场时机（1小时图）

即使大趋势是上涨，也不是随便就买，要等价格回调：

**做多条件**（上涨趋势中）：
1. 1小时图的 EMA20 > EMA100（确认上涨趋势）
2. 当前价格在 EMA20 附近（回调到位）
   - 价格比 EMA20 高 0-1.5%（刚刚回调）
   - 或者价格比 EMA20 高 0-3%，但出现了"看涨反转信号"

**做空条件**（下跌趋势中）：
1. 1小时图的 EMA20 < EMA100（确认下跌趋势）
2. 当前价格在 EMA20 附近（反弹到位）
   - 价格比 EMA20 低 0-1.5%（刚刚反弹）
   - 或者价格比 EMA20 低 0-3%，但出现了"看跌反转信号"

## 三、具体交易流程（举例说明）

### 场景1：BTC 上涨趋势，做多

**第一步：判断大趋势**
- 看4小时图：EMA20 = 50000，EMA100 = 48000
- EMA20 > EMA100 → **大趋势是上涨**

**第二步：等待入场时机**
- 看1小时图：EMA20 = 50500，EMA100 = 49000
- EMA20 > EMA100 → **确认上涨趋势**
- 当前价格 = 50600（比 EMA20 高 0.2%）
- **价格回调到位，可以入场**

**第三步：计算仓位和止损止盈**
- 假设账户有 10000 USDT
- 风险设置：每次交易风险 3% = 300 USDT
- 当前价格：50600
- 止损价：49600（距离 1000 点，约 2%）
- 止盈价：55600（距离 5000 点，约 10%）

**计算仓位**：
- 如果价格跌到止损，最多亏 300 USDT
- 止损距离 = 1000 点
- 每点价值 = 300 / 1000 = 0.3 USDT
- **开仓数量 = 0.3 个 BTC（约 15180 USDT）**

**第四步：下单**
- 方向：做多（买入）
- 数量：0.3 BTC
- 止损：49600（如果价格跌到这里，自动平仓）
- 止盈：55600（如果价格涨到这里，自动平仓）

**第五步：持仓管理**
- 如果价格涨到 51600（盈利 2倍风险），启动移动止损
- 移动止损距离 = 1.5倍 ATR（约 750 点）
- 如果价格继续上涨，止损价会跟着上移
- 如果价格跌回止损价，自动平仓

### 场景2：ETH 下跌趋势，做空

**第一步：判断大趋势**
- 看4小时图：EMA20 = 3000，EMA100 = 3200
- EMA20 < EMA100 → **大趋势是下跌**

**第二步：等待入场时机**
- 看1小时图：EMA20 = 2950，EMA100 = 3100
- EMA20 < EMA100 → **确认下跌趋势**
- 当前价格 = 2960（比 EMA20 高 0.3%，刚刚反弹）
- **价格反弹到位，可以入场做空**

**第三步：计算仓位和止损止盈**
- 账户：10000 USDT
- 风险：3% = 300 USDT
- 当前价格：2960
- 止损价：3060（距离 100 点，约 3.4%）
- 止盈价：2460（距离 500 点，约 16.9%）

**计算仓位**：
- 如果价格涨到止损，最多亏 300 USDT
- 止损距离 = 100 点
- 每点价值 = 300 / 100 = 3 USDT
- **开仓数量 = 3 个 ETH（约 8880 USDT）**

**第四步：下单**
- 方向：做空（卖出）
- 数量：3 ETH
- 止损：3060（如果价格涨到这里，自动平仓）
- 止盈：2460（如果价格跌到这里，自动平仓）

## 四、风险控制机制

### 1. 固定风险

**每次交易最多亏账户的 3%**：
- 账户 10000 USDT → 最多亏 300 USDT
- 账户 50000 USDT → 最多亏 1500 USDT

这样即使连续亏损 10 次，账户也只亏 30%，不会爆仓。

### 2. 止损设置

**止损距离 = 2倍 ATR**（ATR 是价格波动幅度）：
- 如果 BTC 最近波动大（ATR = 500 点），止损距离 = 1000 点
- 如果 BTC 最近波动小（ATR = 200 点），止损距离 = 400 点

**好处**：波动大时止损远一点，避免被正常波动扫止损；波动小时止损近一点，控制风险。

### 3. 止盈设置

**止盈距离 = 5倍风险**（5R）：
- 如果止损距离是 1000 点，止盈距离是 5000 点
- **盈亏比 = 1:5**（亏 1 块，赚 5 块）

**好处**：即使胜率只有 40%，只要盈亏比够大，长期还是能赚钱。

### 4. 移动止损

**当盈利达到 2倍风险时，启动移动止损**：
- 入场价：50000，止损：49000，止盈：55000
- 价格涨到 51000（盈利 2倍风险）
- 移动止损启动，止损价上移到 50250（距离当前价 1.5倍 ATR）
- 如果价格继续涨到 52000，止损价上移到 51250
- **这样即使价格回调，也能保住大部分利润**

### 5. 仓位限制

**每个币种最多用账户的 42%**：
- 账户 10000 USDT → 单个币种最多 4200 USDT
- 如果同时交易 BTC 和 ETH，最多用 8400 USDT

**好处**：分散风险，不会把所有钱押在一个币上。

## 五、策略的优缺点

### 优点

1. **趋势明确时赚钱快**：如果市场有明确趋势，能抓住大部分涨幅
2. **风险可控**：每次最多亏 3%，不会一次亏光
3. **盈亏比好**：亏 1 块赚 5 块，胜率低点也能赚钱
4. **自动止损止盈**：不用盯盘，系统自动管理

### 缺点

1. **震荡行情不赚钱**：横盘时不做交易，可能错过机会
2. **假突破会亏损**：趋势判断错误时，会止损出局
3. **需要耐心**：可能连续亏损几次，才能抓住一次大趋势
4. **需要资金管理**：账户太小（< 1000 USDT）可能无法有效执行

## 六、实际交易效果（回测数据）

根据回测结果（2022-2023年，多个币种）：

- **年化收益率**：约 10-15%（取决于市场环境）
- **最大回撤**：约 15-20%（最坏情况下账户会缩水这么多）
- **胜率**：约 40-50%（10 次交易，4-5 次赚钱）
- **盈亏比**：约 1:3-1:5（亏 1 块，赚 3-5 块）

**关键点**：即使胜率只有 40%，但因为盈亏比是 1:5，长期还是能赚钱。

## 七、总结

**v5.2 策略就是一个"追涨杀跌"的机器人**：

1. **看大趋势**：4小时图判断是涨还是跌
2. **等回调**：1小时图等价格回调到合适位置
3. **固定风险**：每次最多亏 3%
4. **大止盈**：止盈是止损的 5 倍
5. **移动止损**：盈利后保护利润

**适合的市场**：有明显趋势的市场（牛市或熊市）

**不适合的市场**：横盘震荡的市场（会一直观望）

**风险提示**：任何策略都有亏损的可能，建议先在模拟盘测试，确认效果后再用真实资金。
